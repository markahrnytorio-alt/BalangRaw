# LifeEase Abalos Funeral Homes

This project is a NextJS application for Abalos Funeral Homes, providing a platform for clients to avail funeral services. The application allows clients to browse available funeral plans, book services, and manage their bookings. Clients can also subscribe to plans, view their payment history, and update their profile information. The Admin side handles and verifies payments of clients.# BalangRaw
